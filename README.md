# Visionary Prints ‚Äî Multi-Page Website

Visionary Prints is a fully responsive, multi-page website created for a fictional custom printing business. The website showcases products, company information, contact details, and includes interactive features such as a shopping cart, dynamic search, toast notifications, and client-side form submissions.  

The project was built as part of a web development assignment, following best practices for HTML, CSS, JavaScript, responsiveness, accessibility, and front-end design.

Features

Multi-Page Structure
The website includes the following pages:
- **Home**: Hero section, navigation, and featured introduction  
- **About**: Company mission, vision, and background  
- **Products**: Product grid with search functionality and add-to-cart buttons  
- **Checkout**: Dynamic cart, quantity updates, price totals, and order form  
- **Enquiry**: Form for general enquiries, sponsorship, or volunteering  
- **Contact**: Map, locations, and contact form  

Interactive & Dynamic Elements
- **Responsive mobile navigation** with sliding menu animation  
- **Dynamic shopping cart** (add, remove, update quantities)  
- **Live product search** with ‚ÄúNo Results Found‚Äù feedback  
- **Toast notifications** for actions (cart updates, form status, etc.)  
- **Back-to-top button**  
- **Automatic current year** in the footer  
- **Fully simulated server responses** (forms work without a backend)  
- **OpenStreetMap/static map fallback** for full offline compatibility  

Technologies Used
- **HTML5**: semantic structure  
- **CSS3**: external stylesheets, responsive layouts, Flexbox, Grid  
- **JavaScript (ES6)**: dynamic UI updates, events, localStorage, fake server  
- **OpenStreetMap / static image map**: for offline compatibility  
- **No backend required**: all features run on the client side
